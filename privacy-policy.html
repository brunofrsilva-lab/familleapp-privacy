Politique de Confidentialité

FamilleApp — Organisez votre famille, simplement

Mise à jour : avril 2026

💡 En résumé

FamilleApp collecte uniquement les données nécessaires au fonctionnement de l'application. Vos données ne sont jamais vendues ni partagées avec des tiers à des fins publicitaires.

  • ✅ Pas de publicité
  • ✅ Pas de vente de données
  • ✅ Données stockées de façon sécurisée (Supabase)
  • ✅ Suppression complète disponible depuis l'app
  • ✅ Authentification sécurisée via Supabase Auth

👤 Responsable du traitement

FamilleApp est développée et maintenue par Bruno Silva, développeur indépendant basé en Suisse.

Contact : Creator_apps@protonmail.com

🔐 Authentification

FamilleApp utilise Supabase Auth pour sécuriser votre compte. Cela signifie que :

  • Votre mot de passe n'est jamais stocké en clair — il est chiffré par Supabase
  • Votre email est utilisé uniquement pour l'authentification
  • Les sessions sont gérées de façon sécurisée sur votre appareil
  • Chaque famille dispose d'un espace de données strictement isolé

Les politiques de sécurité (Row Level Security) garantissent qu'aucun utilisateur ne peut accéder aux données d'une autre famille, même avec l'URL de la base de données.

🗄️ Où sont stockées vos données ?

Vos données sont hébergées sur Supabase (infrastructure PostgreSQL), dont les serveurs sont situés dans l'Union Européenne (région eu-west).

Supabase est conforme au RGPD. Pour plus d'informations : supabase.com/privacy

Les photos de profil sont stockées dans un espace sécurisé Supabase Storage.

🚫 Partage des données

FamilleApp ne partage jamais vos données personnelles avec des tiers, sauf :

  • Supabase — notre prestataire de base de données (sous contrat de confidentialité)
  • Fournisseurs IA (optionnel) — si vous utilisez le Co-pilote Lumina, vos données de planning (pas votre email) sont envoyées au fournisseur IA que vous avez choisi (Google Gemini, OpenAI ou Anthropic Claude). Vos clés API restent sur votre téléphone.
  • Obligations légales — si requis par la loi

🔔 Notifications

FamilleApp peut envoyer des notifications push locales pour :

  • Rappels de rendez-vous et congés
  • Validation de tâches
  • Anniversaires
  • Statuts de trajet des membres

Ces notifications sont générées localement sur votre appareil et ne passent pas par nos serveurs. Vous pouvez les désactiver dans les Réglages de l'application.

⚖️ Vos droits (RGPD)

Conformément au Règlement Général sur la Protection des Données, vous disposez des droits suivants :

  • Droit d'accès — voir toutes vos données dans l'application
  • Droit de rectification — modifier vos données dans Réglages → Ma Famille
  • Droit à l'effacement — supprimer votre compte depuis Réglages → Supprimer mon profil
  • Droit à la portabilité — contactez-nous pour exporter vos données
  • Droit d'opposition — contactez-nous à Creator_apps@protonmail.com

La suppression de compte depuis l'application efface immédiatement toutes vos données personnelles et, si vous êtes le dernier membre, toutes les données de la famille.

👶 Données des mineurs

FamilleApp est destinée aux familles. Les profils enfants sont créés et gérés par les parents (rôle "Parent") au sein de l'application. Aucun email n'est requis pour les profils enfants.

Nous ne collectons pas sciemment de données personnelles d'enfants de moins de 13 ans de façon directe.
